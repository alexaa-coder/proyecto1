# Cláusula 8 — Operación

Esta sección centraliza todos los procedimientos operativos del SGC, organizados por tipología, así como la gestión de riesgos del producto y la cadena de suministro.

## Contenido

- **Procedimientos generales (PNT 001–030)** — Procedimientos normalizados de trabajo. Cada grupo incluye todas las versiones; las versiones históricas están en `Anexos > Versiones históricas`.
- **Procedimientos específicos (ESP)** — ESP-005 (análisis de riesgos) y ESP-006 (medición de procesos).
- **Gestión de riesgos** — Análisis y matrices de riesgos del SGC y del software VRCardio.
- **Proveedores** — Evaluaciones de proveedores aprobados y documentación de acuerdos.
- **Retirada de producto** — Procedimiento SIM-RP-001 y su versión en inglés.
- **Validación de software** — Informes y protocolos de validación del software VRCardio.
