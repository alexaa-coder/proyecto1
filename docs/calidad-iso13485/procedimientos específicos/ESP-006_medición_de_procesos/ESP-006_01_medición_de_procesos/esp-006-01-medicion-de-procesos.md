# Definiciones y Acrónimos

- **KPIs:** Key Performance Indicator, son parámetros (valores) medibles
 que determinan con qué efectividad se alcanza un objetivo definido.
 sistemática de políticas de gestión de calidad, procedimientos y
 prácticas de tareas para la evaluación, control, comunicación y
 revisión de los riesgos

# Generalidades

El análisis de datos se realizará tomando entradas y salidas de los
procesos claves de SPIKA y se elaborarán indicadores medibles o KPIs
mediante los cuales se podrá medir y tratar estadísticamente los

El análisis de datos incluirá:

- Diseño del producto

- Liberación

- Ventas

En cada indicador se colocará el ratio u objetivo a medir.

# Creacion de un indicador (KPI)

Para la correcta definición e implementación de un KPI se recomienda
seguir los siguientes pasos:

1. Determinar la finalidad: tener una vision del objetivo que se quiere
 lograr en el área.

2. Pregunta KPQ (Key Performance Question): definir preguntas abiertas
 que determinen si el área ha cumplido un objetivo, como por ejemplo:
 ¿que resultados deberían conseguirse?, ¿como puede definirse el
 progreso? ¿por qué es tan importante ese objetivo para el
 área/sistema?

3. Identificar la información y recoleccion de los datos

4. Determinar la frecuencia de medicion

5. Comunicación: al personal afectado, de modo que se asegure la
 efectividad de la definicon, deteccion, medida y seguimiento.

La definición de los KPIs debe ser consistente con el termino SMAT, como
se indica aquí:

| SMART: criterio para la definición de KPIs | |
|--------------------------------------------|------------------------|
| **S** (Specific) | Específicos |
| **M** (Measurable) | Medibles |
| **A** (Achievable) | Conseguibles |
| **R** (Realistic) | Realistas |
| **T** (Time-based) | Definidos en el tiempo |
