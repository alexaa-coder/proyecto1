# Cláusula 5 — Liderazgo

Esta sección recoge los documentos que evidencian el compromiso de la alta dirección con el Sistema de Gestión de Calidad, así como la estructura organizativa formal.

## Contenido

- **Manual de Calidad** — Documento rector del SGC. La versión vigente es la MQ 01.08.
- **Compromiso de la dirección** — Declaración formal de la alta dirección.
- **Puestos de trabajo** — Descripciones de puestos (PDTs) y registros de asignación de responsabilidades (RPTs).
