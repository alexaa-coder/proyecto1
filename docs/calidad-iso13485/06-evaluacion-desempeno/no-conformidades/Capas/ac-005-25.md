Descripción del hallazgo
El SOP PNT-030 para la estrategia de cumplimiento
normativo no describe claramente la identificación de los requisitos
legales pertinentes, la identificación de las fuentes ni las
responsabilidades de las tareas. Además, no se aborda el seguimiento de
las normas CS y/o armonizadas.


Análisis de la causa raíz
El procedimiento PNT-030 no aborda de forma
explícita y estructurada los mecanismos para:
- Identificar los requisitos legales aplicables (MDR, legislación
nacional, etc.).
- Determinar las fuentes oficiales de información (Diario Oficial de
la UE, sitio web de la Comisión Europea, organismos notificados,
asociaciones industriales).
- Asignar responsabilidades claras para el seguimiento de estos
requisitos.
- Supervisar los cambios en las especificaciones comunes (CS) o en
las normas armonizadas pertinentes.
- Esta omisión se debe a que el procedimiento se redactó inicialmente
en una fase temprana del desarrollo del sistema y no se revisó tras la
entrada en vigor completa del MDR.


Corrección inmediata
Registrar la no conformidad en el sistema de calidad y
proporcionar la trazabilidad del seguimiento.


Acción correctiva / preventiva
Revisión del procedimiento PNT-030, incluyendo:
- Sección específica para la identificación de los requisitos legales
y reglamentarios aplicables, incluyendo el MDR, las directivas
aplicables, la legislación nacional y las directrices del MDCG.
- Inclusión de una lista de fuentes oficiales de información
actualizada (sitios web, boletines, bases de datos de normas, etc.).
- Asignación clara de las partes responsables de supervisar, revisar
y actualizar estos requisitos.
- Descripción del proceso de supervisión de las normas armonizadas y
las especificaciones comunes (CS), incluyendo la frecuencia, las
herramientas de supervisión (por ejemplo, notificaciones del CEN, DOUE)
y los criterios para evaluar su aplicabilidad.
- Integración con el procedimiento de evaluación de riesgos y gestión
de cambios para garantizar que el sistema se actualice cuando cambien
los requisitos.

Evaluacion de impacto:
La no conformidad no tiene un impacto directo en el
producto ni en la seguridad del mismo, pero sí compromete la capacidad
del sistema para adaptarse a los cambios normativos con garantías. Las
medidas correctivas establecerán un sólido sistema de vigilancia
normativa para garantizar el cumplimiento continuo del MDR y otras
normativas aplicables.
