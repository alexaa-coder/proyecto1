# Sistema de Gestión de Calidad — ISO 13485

## Objeto y campo de aplicación

Este sistema de gestión de calidad ha sido diseñado e implantado por **Spika Tech** para el diseño, desarrollo, producción y comercialización del producto sanitario **VRCardio**, de conformidad con los requisitos de la norma **UNE-EN ISO 13485:2016** y el **Reglamento (UE) 2017/745** sobre productos sanitarios (MDR).

## Estructura de esta sección

La documentación está organizada siguiendo las cláusulas principales de la norma:

| Sección | Contenido |
|---|---|
| 01 — Contexto de la organización | Contexto, certificados, licencia previa, normas de referencia |
| 02 — Liderazgo | Manual de Calidad, compromiso de la dirección, estructura de puestos |
| 03 — Planificación | Objetivos de calidad, recursos y plantillas |
| 04 — Soporte | Formación del personal, gestión de PNTs |
| 05 — Operación | Procedimientos generales y específicos, gestión de riesgos, proveedores |
| 06 — Evaluación del desempeño | Auditorías, métricas, no conformidades, revisión por la dirección |
| 07 — Mejora | Acciones de mejora continua |
| Anexos | Guía de uso, versiones históricas del Manual de Calidad |

## Referencias normativas principales

- UNE-EN ISO 13485:2016 — Productos sanitarios. Sistemas de gestión de la calidad.
- Reglamento (UE) 2017/745 — Reglamento de productos sanitarios (MDR).
- ISO 14971:2019 — Gestión de riesgos para productos sanitarios.
- IEC 62304:2015 — Software de dispositivos médicos.
