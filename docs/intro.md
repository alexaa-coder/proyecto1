# Portal de Documentación Normativa — VRCardio / Spika Tech

Este portal centraliza toda la documentación normativa de Spika Tech relacionada con el producto sanitario **VRCardio**, organizada en torno a dos sistemas de gestión certificados:

- **ISO 13485:2016** — Sistema de Gestión de Calidad para productos sanitarios.
- **ISO/IEC 27001:2022** — Sistema de Gestión de Seguridad de la Información.

## Cómo usar este portal

Utiliza la barra lateral izquierda para navegar por las secciones de cada norma. Cada sección sigue la estructura de cláusulas del estándar correspondiente, desde el contexto organizativo hasta la mejora continua.

Los documentos están disponibles en español. Las versiones en inglés se indican con el sufijo `_EN` en el nombre del archivo.
