# Cláusula 8 — Operación

Esta sección centraliza todos los procedimientos operativos del SGSI, la gestión de la continuidad del negocio y las actividades de seguridad en el desarrollo de software.

## Contenido

- **Procedimientos (PRO 001–023)** — Procedimientos de gestión de requisitos legales, roles, documentación, riesgos, activos, infraestructura, acceso, backups, incidentes, auditoría, proyectos IT, cambios TIC, redes, identidades e inteligencia de amenazas.
- **Continuidad del negocio** — Plan de continuidad y recuperación ante desastres (CDN-001), procedimiento de copias de seguridad (CDN-002), BIAs.
- **Seguridad del desarrollo** — Guías de auditoría de código, reportes del Code Checker, protocolo y hallazgos de pentesting.
- **Inteligencia de amenazas** — Guías CCN y referencias INCIBE relevantes para el sector salud.
- **Protección de proyectos IT** — Cláusulas de protección y plantillas SLA.
- **Proveedores** — Evaluaciones de proveedores IT (Atlasian, Born, Google, HTCert, Konexio, Microsoft, SmartMDR, Unity).
